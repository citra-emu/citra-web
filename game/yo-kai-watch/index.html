Summary

Yo-Kai Watch runs well enough on Citra to be played, with the only detrimental issue being a gradual slowdown as the game is played for extended periods of time. This can be remedied by simply restarting emulation. The game has a bug for not displaying semi-transparent yo-kai (as in yo-kai lens or when you befriend them) if not in native resolution.
